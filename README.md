# CodeAlpha Iris Flower Classification

## Project Overview

This project classifies Iris flowers into three species:

* Setosa
* Versicolor
* Virginica

using Machine Learning with Scikit-learn.

## Technologies Used

* Python
* Scikit-learn
* Pandas
* Matplotlib

## Steps Performed

1. Loaded Iris dataset.
2. Split data into training and testing sets.
3. Trained a Random Forest Classifier.
4. Evaluated model performance.

## Results

Accuracy: 100%
